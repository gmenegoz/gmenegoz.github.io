Istruzioni per la Configurazione

  1. Vai alla Google Cloud Console
  2. Crea un nuovo progetto o seleziona uno esistente
  3. Abilita la Google Sheets API
  4. Crea credenziali OAuth 2.0:
    • Tipo: "Web application"
    • Authorized JavaScript origins: aggiungi il tuo dominio (es. http://localhost:8080)
  5. Copia il Client ID nel file js/config.js
  6. Inserisci l'ID del tuo Spreadsheet nel file js/config.js
  7. Ricarica questa pagina
